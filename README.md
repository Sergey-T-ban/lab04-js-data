# Практична робота №9–10

## Тема

Асинхронна робота з даними у JavaScript: fetch(), JSON, рендеринг каталогу та керування станом інтерфейсу.

## Мета роботи

Навчитися працювати з асинхронним отриманням даних за допомогою fetch(), використовувати JSON-файли як джерело даних, реалізовувати динамічний рендеринг контенту, пошук, фільтрацію, сортування, пагінацію та збереження даних у localStorage.

## Виконані завдання

### 1. Підготовка JSON-даних

Створено файл `data/items.json`, який містить масив із 10 об'єктів. Кожен об'єкт має поля:

* id
* title
* category
* description
* image
* level
* rating
* price
* featured

### 2. Асинхронне завантаження даних

Для отримання даних використано функцію `fetch()` та конструкцію `async/await`.

### 3. Реалізація станів інтерфейсу

Передбачено:

* Loading State (під час завантаження даних);
* Error State (у випадку помилки завантаження);
* Empty State (якщо результатів не знайдено).

### 4. Динамічний рендеринг карток

Картки каталогу генеруються автоматично на основі даних із JSON-файлу за допомогою JavaScript.

## Висновок

У ході виконання практичної роботи було освоєно роботу з асинхронним отриманням даних за допомогою Fetch API, реалізовано динамічний каталог на основі JSON-файлу, пошук, фільтрацію, сортування, пагінацію та збереження обраних елементів у LocalStorage.
